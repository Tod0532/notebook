## 📋 项目概述

**项目名称**：动计笔记 (MovePlan Memo) - AI智能教练版

**项目定位**：一款融合**日常记事**、**智能提醒**、**运动追踪**、**AI教练**的全能生活管理APP。通过AI生成个性化饮食+训练计划，结合心率监测技术，帮助用户科学健身、达成目标。

**目标用户**：
- 希望科学健身的上班族、学生
- 需要个性化饮食+训练计划的健身爱好者
- 希望通过数据监测提升运动效果的用户
